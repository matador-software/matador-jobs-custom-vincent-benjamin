== Description ==

Custom Matador Jobs Pro extension for Vincent Benjamin Group.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the '/wp-content/plugins/matador-jobs-custom-vincent-benjamin' directory, or go to the WordPress plugins screen directly on your site and click "Add Plugin".
1. Activate the plugin through the 'Plugins' screen in WordPress
